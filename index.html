The Problem

When speaking isn't an option

Millions of people face life-threatening situations where calling for help isn't possible — not because they won't, but because they can't. Traditional emergency systems were never designed for silence.

Whether it's a kidnapping, domestic abuse, or a threatening situation — every second of delay matters.

🚗
Kidnapping / Abduction
Victim cannot speak or unlock phone without drawing attention from the abductor.
🏠
Domestic Abuse
Survivor is monitored closely; any visible call or message attempt can escalate danger.
🌃
Harassment / Night Travel
Person being followed cannot safely pull out phone to make a visible emergency call.
Why Current Solutions Fail

Built for the wrong situation

🔒

Emergency dialers require the phone to be unlocked first — impossible if the attacker is watching or controlling the device.

🔴

🎙️

Our Solution

Invisible. Instant. Life-saving.

SilentShield is a mobile application that hides inside a functional, convincing calculator app. The user triggers an SOS silently — the system does the rest without any visible indication.

  • Triggers via hidden gesture: secret code entry, shake pattern, or tap sequence
  • Runs completely in the background — no screen flash, no sound
  • Sends live GPS location and SOS alert to pre-selected contacts and emergency services
  • Continues functioning as a normal calculator if the attacker checks the phone
Key Features

Designed for the most extreme conditions

🧮

Hidden Trigger Methods

Three covert activation methods: enter a secret numeric code in the calculator, perform a custom shake pattern, or use a specific tap sequence — all invisible to an observer.

📍

Live Location Tracking

The moment SOS is triggered, the system locks onto GPS and begins streaming your real-time location to emergency contacts — even as you move.

🔕

Completely Silent Alerts

No vibration. No sound. No screen change. The entire alert dispatch happens invisibly in the background — the phone looks completely normal.

🎭

Fake UI for Safety

The app is disguised as a fully functional calculator. Anyone who picks up the phone sees only a standard calculator — zero indication of an active emergency.

Technical Deep Dive

System Architecture & Implementation

A complete breakdown of how SilentShield is engineered — from trigger detection to alert delivery.

Architecture

Four core modules

📱

Mobile Application

Built in Android (Java/Kotlin). Houses the fake calculator UI, background trigger detection service, and local state management. Runs as a persistent foreground service.

☁️

Backend (Firebase / AWS)

Firebase Realtime DB stores emergency contact data and session state. AWS Lambda + SNS optionally handles high-priority SMS dispatch and serverless alert routing.

🚨

Alert Dispatch System

Multi-channel alert delivery: silent push notification via Firebase Cloud Messaging + SMS fallback via Twilio/AWS SNS. Ensures delivery even in low-connectivity areas.

📍

Location Tracking Module

Google Maps API + Android Fused Location Provider for high-accuracy GPS. Live coordinates are streamed to the backend every 15 seconds and shared via a secure link.
